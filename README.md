# archer-cms
Fullstack MasterClass github

## Requirements:
Mongo 3.X or later
NodeJs with NPM

## Steps:

1) clone repo
2) run npm install
3) have a look at import.js create any other intitial users. Note that some are needed to run the system
4) run node import.js
